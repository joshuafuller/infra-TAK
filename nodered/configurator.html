ATAK Network Link URL

Check which severity levels to include. Polygon geometry is most common for Extreme and Severe.

Comma-separated 2-letter state codes, e.g. CA,OR,WA. Leave blank for all US states.

Leave blank to use the NAPSG Public Alert symbol CDN. Override only if self-hosting icons.

ATAK setup: Overlay Manager → + → Add URL → paste the URL above → set refresh to 5 min → OK.
Alert polygons are colored by severity (🔴 Extreme, 🟠 Severe, 🟡 Moderate, 🔵 Minor). Icons use NAPSG Public Alert symbols (CC BY 4.0).
